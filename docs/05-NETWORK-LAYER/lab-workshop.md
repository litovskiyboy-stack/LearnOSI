# 🧪 Лабораторная работа: Сетевой слой (Layer 3)

## 🎯 Цель работы

Изучить работу сетевого слоя, научиться работать с IP-адресацией, подсетями и маршрутизацией.

---

## 🔬 Задание 1: Узнай свой IP-адрес

### **Шаг 1.1 — Получи информацию о сетевых интерфейсах**

#### **Linux/Mac:**
```bash
ip addr show
# или
ifconfig -a
```

**Что нужно найти:**
- Твой IP-адрес (например, `192.168.1.100`)
- Маска подсети (например, `/24` или `255.255.255.0`)
- Название интерфейса (eth0, wlan0 и т.д.)

#### **Windows:**
```cmd
ipconfig /all
```

**Что нужно найти:**
- IPv4 Address
- Subnet Mask
- Default Gateway
- DNS Servers

## 🔬 Задание 8: Настрой статический маршрут (опционально)

### **Шаг 8.1 — Добавь статический маршрут**

#### **Linux:**
```bash
# Добавить маршрут к конкретной сети:
sudo ip route add 10.0.0.0/8 via 192.168.1.254 dev eth0

# Показать таблицу маршрутов с новым маршрутом:
ip route show
```

#### **Windows:**
```cmd
# Добавить маршрут к конкретной сети:
route -p add 10.0.0.0 mask 255.0.0.0 192.168.1.254 metric 30

# Показать таблицу маршрутов с новым маршрутом:
route print
```

---

### **Шаг 8.2 — Удали статический маршрут**

#### **Linux:**
```bash
sudo ip route del 10.0.0.0/8 via 192.168.1.254 dev eth0
```

#### **Windows:**
```cmd
route delete 10.0.0.0
```
